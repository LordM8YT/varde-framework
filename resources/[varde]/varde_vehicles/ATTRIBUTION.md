# Garage coordinate attribution

The default public parking labels and in-game access/spawn coordinates in
`config/vehicles.json` were transcribed from
[Qbox-project/qbx_garages `config/server.lua`](https://github.com/Qbox-project/qbx_garages/blob/969c29d7396adc126183aeaf880688aae9888cf7/config/server.lua)
at commit `969c29d7396adc126183aeaf880688aae9888cf7`.

That upstream project is distributed under GPL-3.0. Varde uses only the factual
GTA V world locations requested for familiar defaults. No Qbox Lua source,
garage algorithm, database schema, UI, API, or runtime dependency is included.
Varde's garage implementation and JSON schema were independently written.
